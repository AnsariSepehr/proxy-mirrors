# Son Proxy Mirrors

A clean and modern mirror directory for **Nexus Repository Manager**.

Live Demo: [https://nexus.son.ir/mirrors/](https://nexus.son.ir/mirrors/)

## Features

- Beautiful card interface with dark & light theme support
- Click any mirror → detailed panel with ready-to-copy URLs and configs
- One-click copy buttons
- Horizontal scrollable code blocks (no wrapping)
- Search functionality
- Smooth hover animations (tilt + light sweep)
- Fully responsive
- Maintainer card with links
- Small animated bat at the bottom

## How to Use This Website

1. Open the website
2. Browse or search for the mirror you need
3. Click on any card (Composer, Maven, NPM, etc.)
4. The right panel opens with:
   - The mirror URL
   - Ready-to-use configuration code
   - Copy buttons
5. Click **Copy ↑** to copy the URL or code snippet
6. Paste it into your `composer.json`, `settings.xml`, `pip.conf`, etc.
